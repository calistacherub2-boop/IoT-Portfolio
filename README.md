# IoT Portfolio

## 1) Real-time Environmental Monitoring System
### Project Details
A system designed to monitor environmental parameters such as temperature, humidity, and air quality in real-time.

### Technologies Used
- Sensors (DHT11, MQ-135)
- Microcontroller (Arduino)
- Cloud Service (Firebase)

### Roles
- Developer: Implementing the sensing and cloud integration.

### Features
- Real-time data display on web interface.
- Alerts for unusual environmental conditions.

### Status
- Completed

### How to Run Instructions
1. Connect the sensors to the Arduino.
2. Upload the code to the Arduino.
3. Configure the cloud service.
4. Open the web interface to monitor data.

## 2) Smart Accident Detection and Auto Alert System
### Project Details
A system that detects accidents through accelerometer data and sends alerts to emergency contacts.

### Technologies Used
- Sensors (Accelerometer)
- Microcontroller (Raspberry Pi)
- SMS Gateway

### Roles
- Developer: Building the detection algorithm.

### Features
- Automatic alerts to contacts upon accident detection.
- Location tracking via GPS.

### Status
- In Progress

### How to Run Instructions
1. Set up the Raspberry Pi with the necessary libraries.
2. Integrate the accelerometer and GPS.
3. Run the detection script.

## 3) IoT-Driven Smart Worm Automation System
### Project Details
A system for managing worm farming automatically including feeding and environmental control.

### Technologies Used
- Microcontroller (Arduino)
- Sensors (Moisture, Temperature)
- Actuators for feeding

### Roles
- Developer: Overseeing the automation process.

### Features
- Automated feeding schedule.
- Environmental monitoring for optimal worm growth.

### Status
- Planned

### How to Run Instructions
1. Set up sensors and actuators.
2. Code the feeding schedule into the Arduino.
3. Monitor via the connected application.
